# Strategia komunikacji TKOPD

Repozytorium zawiera skrypt Google Apps Script do utworzenia ankiety Google Forms na podstawie dokumentu:

`Ankieta_dla_zespolu_TKOPD_strategia_komunikacji.docx`

## Jak utworzyc formularz

1. Wejdz na https://script.google.com/.
2. Utworz nowy projekt.
3. Wklej zawartosc pliku `google_apps_script/create_tkopd_form.gs`.
4. Uruchom funkcje `createTKOPDCommunicationSurvey`.
5. Zaakceptuj uprawnienia Google.
6. Link do gotowej ankiety pojawi sie w `Execution log`.

Skrypt tworzy formularz Google Forms z 10 sekcjami i 43 pytaniami.
